Что такое HTML?

HTML (HyperText Markup Language) — это стандартный язык разметки, используемый для создания структуры веб-страниц. Он определяет, как элементы должны располагаться и отображаться в браузере.

Где используется HTML?

  • Создание структуры веб-сайтов
  • Вёрстка текста, изображений, таблиц и форм
  • Встраивание мультимедиа и скриптов
  • Создание семантической структуры документа

Важные теги HTML

  • <html> — корневой элемент страницы
  • <head>, <title>, <meta> — служебная информация
  • <body> — основное содержимое
  • <h1>-<h6>, <p>, <a> — текстовая разметка
  • <img>, <video>, <form> — мультимедиа и формы

Пример HTML-кода

<!DOCTYPE html>
<html lang="ru">
  <head>
    <meta charset="UTF-8">
    <title>Пример</title>
  </head>
  <body>
    <h1>Привет, мир!</h1>
    <p>Это мой первый HTML-документ.</p>
  </body>
</html>
